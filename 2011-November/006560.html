La wiki no esta instalada, en lugar de la url de berlios y el nick de
berlios, debes usar la url que te indique en gitorious, y tu nick
correspondiente.

Para instalarlo desde las fuentes, es tan sencillo como hacer lo
siguiente, por ejemplo:

git clone
ssh://VUESTRO_NICK_EN_GITORIOUS@git@gitorious.org:bulmages/bulmages.git
<http://VUESTRO_NICK_EN_GITORIOUS@git@gitorious.org:bulmages/bulmages.git>
cd bulmages
mkdir build
cd build
cmake ../bulmages
make
sudo make install
