>  Hola Tomeu.
>
> Los pasos son los siguientes:
>
> 1) Entra en tu directorio GIT y ejecutas "git remote -v" para ver el
> listado de repositorios remotos configurados. Verás que te sale el de
> Berlios con el alias "origin".
>
> 2) Borra el repositorio de Berlios (origin) con "git remote rm origin".
>
> 3) Añade el nuevo repositorio de Gitorious con "git remote add -t master
> origin git en gitorious.org:bulmages/bulmages.git".
> (Poner -t master es para sólo sincronizar con esa rama).
>
> 4) Sincroniza con el nuevo repositorio con "git pull origin master".
>
>
> Si todo va bien se hará un "merge" con tu copia local y luego ya puedes
> hacer "git commit -a" y "git push" de forma normal.
>
>
> Un saludo.
