(I don't know if Henry -Aishen- is reading this, I'm writing in
English just in case).

I cannot compile bulmages in Ubuntu 11.10.

This distribution has pushed forward the Debian changes for
"multiarch" support and now the configuration of bulmages cannot find
the QT libraries. Debian "Wheezy" will have multiarch support and
that's why it has been introduced in Ubuntu.

This is a very nice change allowing for instance to convert a 32bit
system into a 64 bit system without reinstalling, but on a more
practical matter making the --force-architecture flag unnecessary

I'm looking around but I haven't been able to find anything on what
needs to be changed in configuration files. A part from copying  all
the libraries from "/usr/lib/x64_86/ to /usr/lib I cannot think of
anything else right now, but that is an awful hack.

More on this:

Debian:
http://wiki.debian.org/Multiarch
(the section Multiarch conversion howto,
http://wiki.debian.org/Multiarch/Implementation, looks promising)

Ubuntu:
https://wiki.ubuntu.com/MultiarchSpec
