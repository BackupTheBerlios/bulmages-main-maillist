Hola

Deberiamos distinguir entre un problema de bgtrml2pdf y uno de bulmafact.

Si has intentado imprimir la factura en el directorio ~/.bulmages tienes que 
tener un archivo factura.rml 

Primero comprueba que el archivo tiene contenido (formato RML) y que no está 
vacio.

A partir de este archivo se genera el PDF usando el comando bgtrml2pdf 
factura.rml > factura.pdf

Entonces si falla te escupirá un error y nos podras comentar el error y 
adjuntar el archivo rml para que lo podamos probar nosotros.

Si la factura es real cambia los datos por privacidad.

Salut
