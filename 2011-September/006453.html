Para aclarar los mensajes que llegan a la lista con remitente
admin en berlios.de y texto de errores sin ninguna aportación adicional:

Cada vez que se cambia cualquier cosa en un bug report, por ejemplo se
cambia una etiqueta de new a fix released, el sistema genera un email
a la lista. Así que cuando hacemos limpieza de bugs o feature
requests, cerrándolos o cambiando etiquetas, se verá un mensaje.
