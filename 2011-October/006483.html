Hola Leo.

Si, segur que hi ha un miracle i no s'ha de migrar a Gitorious, pero per
ara l'ordre és d'abandonar el vaixell :)

I tenint en compte que l'unic que s'ha de fer és donars-hi d'alta i
executar:

git remote add gitorious git en gitorious.org:bulmages/bulmages.git
git remote rm origin

(o tornar-ho a baixar)

No veig que sigui traumatic. I vull donar de baixa el Git de Berlios per
evitar duplicitats i tenir una part de la feina feta.

Salut!
