Imagin que el que passa es que has fet un update i un make install sense 
esborrar el /usr/local/share/bulmages que tenies abans i que han quedat 
combinades les dues versions.

Prova a esborrar el esmentat directori i després fes un make install de nou.

Sobre lo dels plugins que queden marcats es normal ja que els plugins que 
toquen la base de dades hi deixen una marca que el bulmasetup detecta i emplea 
per determinar si un plugin esta instalat o no. Això es així perque es pot 
donar el cas de algun plugin que no s'hagi d'instalar com a tal i que l'unic 
que faci siguin canvis a la base de dades.

No has de patir perque surtin marcats com a instal·lats.
