Creo que debes olvidarte de ese repositorio. Ya no se utiliza 
subversion, sino git. Haz una copia del repositorio con:

$ git clone --branch release --single-branch 
git://gitorious.org/bulmages/bulmages.git

Previamente instala git-core, si no lo tienes. Los demás pasos están 
bien, aunque sobra un make install.

Suerte :)
