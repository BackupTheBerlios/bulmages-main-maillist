Hola a todos,

Como no lo pone en la guía de la wiki, y puede que sea interesante para muchos de vosotros.

Podéis realizar una plantilla para los commits de git fácilmente, tan solo hay que escribir un documento con el texto deseado para que se replique en nuestros commits. Por ejemplo yo tengo:

--
Daniel Ripoll Osma
www.danielripoll.es
6688.27.4.90

Web&  Desktop Developer


Con dos saltos de linea antes del "--".

Y a continuación

git config --global commit.template PLANTILLA_FUERA_DEL_REPOSITORIO

Así cada vez que haga git commit en cualquier repositorio, me escribe esa plantilla, muy útil para dejar nuestra huella, y no solo un simple email, dentro de nuestros commits.

Si no queremos que lo haga por todos los repostorios, y tan solo en uno, quitamos la particula "--global".


Eso es todo, espero que aunque no sea un consejo de programación, os sea útil.

Happy coding y felices fiestas de Saturnalia!
