Bones a tothom:

Con el AutoForms funcionando, creo que ahora lo más interesante es poder
instalarlas (como si de un plugin se tratase).
Además que no hay ningún ejemplo de su uso de fácil acceso.

Asi que si no hay nada más urgente, y Daniel no lo migra a C++ antes, me
voy a dedicar a que desde bulmasetup se puedan instalar plugins hechos
integramente con autoforms.
Analizando el tema me salen principalmente tres cosas:

- Ampliar los README para que soporten nuevas directivas de instalación.
- Permitir la generación dinámica del autoform_XXXXXXX_spec.spc (juntando
varios módulos de autoforms). O tal vez mejor sería permitir algo del
estilo autoform_XXXX_YYY_spec.spc para no mezclar un plugin con otro. Esta
segunda opción requiere modificar el código del plugin, pero es más útil.
- Buscar una buena ubicacion para los autoforms de una empresa. Supongo que
/opt/bulmages/autoform_XXXX es un buen sitio si los autoforms van a ser
modificables. Aunque otra buena opción es dejarlos en el share y no
duplicarlos por cada empresa.
