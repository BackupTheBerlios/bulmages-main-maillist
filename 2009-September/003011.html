Cuando un usuario no supervisor de PostgreSQL arranca BulmaFact y actualiza la 
lista de empresas, aparece un error si activamos la depuración:
BlDbRecordSet::BlDbRecordSet: Error en la consulta: SELECT * FROM pg_tables 
WHERE tablename = 'configuracion'

Si el mismo usuario se conecta por psql a la base template1, la misma consulta 
funciona.

Dicho usuario también puede acceder a la base de datos de la empresa si usamos 
el parámetro -d: bulmafact -d NOMBRE_BD
