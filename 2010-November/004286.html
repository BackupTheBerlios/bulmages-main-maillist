Després del canvi de posar el control per obrir el
formulari d'alumnes directament des del formulari de
pares/mares/tutors, al camp d'alumne sols té un espai i
surt el primer nom de l'alumne.

No sé si és la millor opció, potser el millor seria
veure més camps:

Nom alumne 1/ Primer Cognom / Segon Cognom / Classe /
Control formulari / esborrar

Nom alumne 2/ Primer Cognom / Segon Cognom / Classe /
Control formulari / esborrar

Nom alumne 3/ Primer Cognom / Segon Cognom / Classe /
Control formulari / esborrar

Nom alumne 4/ Primer Cognom / Segon Cognom / Classe /
Control formulari / esborrar

Crec que també estaria bé que l'usuari tingués dues
opcions per entrar dades de l'alumne. O bé entrar les
dades directament aquí a aquesta formulari, o bé clicar
"Control formulari" i obrir el formulari de l'alumne i
entrar dades més completes. 

També estaria bé que si desa la informació directament
al formulari de pares/mares, per defecte camps de
l'alumne com Adreça / Telèfons / email, quedessin
omplerts amb les dades del mare/pare/tutor. L'usuari
sempre tindria l'opció de canviar aquesta informació
per alguna d'específica de l'alumne si ho vol fer,
entrant al formulari complet de l'alumne.
