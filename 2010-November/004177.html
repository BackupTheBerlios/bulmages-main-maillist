Summary: La ventana no se maximiza con NX sobre Windows

Details: Cuando lanzamos el progama desde una sesión NX en Windows y cerramos con la ventana maximizada, el siguiente arranque muestra la ventana sin maximizar.
La ventana ocupa todo el espacio menos un espacio del tamaño de la barra de título en la parte superior, resultando confuso a primera vista, ya que si teníamos otra ventana abierta parece que Bulmagés tenga dos barras de título.

Follow-Ups:

Date: 2010-Nov-03 18:35
By: hilex

Comment:
Para corregir el desplazamiento vertical que va acumulándose al lanzar el programa mediante NxClient, podemos usar un script que "truca" la configuración justo antes de ejecutar bulmafact:
sed -r -i 's/<Y>.*<\/Y>/<Y>-4<\/Y>/' ~/.bulmages/bulmafact_{bd}.cfn"
sed -r -i 's/<X>.*<\/X>/<X>-4<\/X>/' ~/.bulmages/bulmafact_{bd}.cfn"
-------------------------------------------------------

Comment:
Bueno, probando la versión compilada en Windows veo que la posición ya es correcta (no se queda maximizada, pero como si lo estuviera).

Donde veo que no se posiciona bien es al usarlo mediante NxClient: cada vez que lo abro la ventana baja un poco más. Será problema del Nx.
