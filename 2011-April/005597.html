Mistake:
If you have downloaded as anonymous and you are a developer, you must use:

git clone http://YOURUSER@git.berlios.de/gitroot/bulmages
cd bulmages
git init
git push ssh://youruser@git.berlios.de/gitroot/bulmages master

It's easier to do:

git clone ssh://girardhenri@git.berlios.de/gitroot/bulmages
git push
