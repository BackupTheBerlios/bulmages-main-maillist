- Para más claridad, Git puede usar colores en su salida. Para ello se puede 
ejecutar una vez:
        git config --global color.diff auto
        git config --global color.status auto
        git config --global color.branch auto

- Una orden bastante útil es "whatchanged", que muestra qué ha cambiado en un 
archivo en particular:
         git whatchanged archivo_en_particular
