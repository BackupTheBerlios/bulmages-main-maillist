Gràcies, ara que sé com es compila i s'engega la versió per a linux
posaré unes instruccions a la web de la FaPaC perquè hi ha força gent
esperant a provar-ho a Linux i poder fer aportacions. 

Postser hauríem de repassar les dependències per compilar-lo, jo tinc
aquestes:

sudo apt-get install build-essential cmake-gui translate-toolkit lokate
sudo apt-get install qt3-assistant qt4-designer qt4-qtconfig qt4-dev-tools
sudo apt-get install postgresql-8.3 postgresql-client libqt4-sql-psql postgresql-doc-8.3
sudo apt-get install python-minimal python-ooolib python-qt4 python-reportlab python-psycopg2 gettext

Però també està el tema d'afegir les llibreries /usr/local/lib a
/etc/ld.so.conf.d/i486-linux-gnu.conf 

i no se si em deixo alguna cosa. 

També és imminent que el Tomeu acabi una altra versió més flexible amb
el colinux, perquè la primera sols la van poder fer córrer dos o tres
persones (tot i que jo la vaig fer córrer a la primera),
