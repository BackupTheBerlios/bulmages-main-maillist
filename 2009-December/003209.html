¿Hay algo implementado en Bulmages que teniendo por ejemplo la cadena "26" nos 
genere "00026"?

En caso de que no, ¿puedo agregar esta función en blfunctions.h?
Es para poder usarla en varios plugins que estoy haciendo, y no sé como 
compartir una clase de funciones comunes a dichos plugins.


QString anteponerCeros( QString num, int digis )
{
   QString retorno;
   int tam = num.length();

   if ( tam > digis ) {
      retorno = num;
   }
   else {
      retorno = QString( digis - tam, '0' );
      retorno += num;
   }

   return retorno;
}
