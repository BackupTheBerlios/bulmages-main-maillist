Hola a todos

>> Guardar imágenes de artículos

Efectivamente, para que se guarden las imágenes había que editar el archivo
/etc/bulmages/bulmafact.conf
Ahí salen las siguientes directivas

CONF_DIR_IMG_ARTICLES     /var/bulmages/articles/
CONF_DIR_IMG_PERSONAL   /var/bulmages/personal/

Como ninguno de las dos carpetas existían en mi kubuntu, las he creado y
asunto resuelto.
Decir que las he creado dándoles el mismo user que lanza la aplicación, no
como root.
Las imágenes de artículos se guardan en la carpeta articles (artículos).
Ignoro para qué servirá la otra pero ya está creada para no tener problemas.
