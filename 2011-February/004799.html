>>> Buenas, paso por aqui la chuleta para desinstalar completamente una
>>> empresa. ($$basededatos$$) es la empresa a eliminar.
>>>
>>> 1.- Eliminar la base de datos
>>>
>>> dropdb $$basededatos$$
>>>
>>> 2.- Eliminar las plantillas de impresion
>>>
>>> rm -rf /opt/bulmages/openreports_$$basededatos$$
>>>
>>> 3.- Eliminar el archivo de configuracion
>>>
>>> rm /etc/bulmages/*$$basededatos$$.conf
>>>
>>> Y con esto creo que queda completamente desinstalada la empresa.
>>>
>>> Opcionalmente borrar los archivos de configuracion
>>>
>>> rm ~/.bulmages/*$$basededatos$$*
>>>
>>>
>>> Espero que sea util
