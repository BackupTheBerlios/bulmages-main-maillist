> I remenber that we have not translations of descriptions for every
> plugin in all the project. (The text that appears in bulmasetup)
> 
> This is possible to do that but is a little hard work.
> 
> In first place we need to make a file README.fr in every plugin and
> with a syntax equal to README.es. This file is get by bulmasetup to
> know what install in every company.
> 
> After that we need to change all CMakeLists.txt to install translated
> files and finally we have to modify bulmasetup to get README.fr in
> case of french locales.
> 
> The most important thing is to respect the syntax in README because
> bulmasetup parses the file and gets is contents. An error will crash
> the application.
