He cambiado el diseño de la consulta para verla mejor.
Está ingeniosa, no veo que se pueda simplificar mucho ya, a no ser que exista alguna función de PostgreSQL que no conozcamos.

SELECT
	*,
	Fecha1 + "S"."I" AS Fecha2
FROM  (
	SELECT
		('01/07/2011'::DATE + "Series"."Index") AS Fecha1,
		EXTRACT ('dow' FROM '01/07/2011'::DATE + "Series"."Index" ) AS dowe
	FROM generate_series(-6,2) AS "Series"("Index")
	WHERE EXTRACT ('dow' FROM '01/07/2011'::DATE + "Series"."Index" ) = 1 LIMIT 1
) AS S2, generate_series(0,41) AS "S"("I");
