Agrego un nuevo e interesante plugin.
Este lanza los parsers de metatags para las plantillas RML sobre plantillas python que generan
informes de openoffice mediante ooolib. De esta forma podemos programar informes de hoja de calculo
sin necesida de crear un plugin para ellos.
Junto con el plugin adjunto una plantilla .pys de ejemplo que muestra un poco la capacidad de este tipo de informes.
Tengo ganas de ver donde meten los empaquetadores las plantillas pys ya que son al mismo tiempo una plantilla
de documento y un programa en python. JUAAAAAA JUAAAAAAAAAAAAA JUUUUAAAAAAAAAAAAAAAAAAAAAAA
