Bien, esto parece que me toca responderlo a mí. Lo primero de todo, 
disculpad por el retraso. Y ahora, al tema.

Hace unos dos años, desarrollamos para Bulmafact un plugin que pudiera 
exportar las facturas a formato UBL ( 
http://en.wikipedia.org/wiki/Universal_Business_Language ) e importar 
las facturas desde este formato. Esta parte es funcional. Lo que no 
fuimos capaces de solucionar fue la integración de un programa o el uso 
de alguna API para firmar dichas facturas o verificar la validez de las 
firmas de las facturas a importar.

Lo intentamos mucho con gpgsm ( http://linux.die.net/man/1/gpgsm ) pero 
en ese momento no estaba preparado para producción y nos daba problemas 
con los certificados de la FNMT.

Es una lástima que no tuviéramos antes las referencias de 
http://www.facturae.es
Por lo visto han cogido el UBL y se han hecho ellos un esquema derivado. 
A ver si ahora conseguimos compatibilizarlo y con la web que indicó 
Francisco Javier ( http://sinadura.net ) arreglamos el problema con la 
firma.

La verdad es que me ha dado mucha rabia encontrarme con facturae.es a 
estas alturas y no cuando nos pusimos a desarrollar el plugin de efactura.

Espero haberte contestado en algo, Alfredo.

Un saludo.
