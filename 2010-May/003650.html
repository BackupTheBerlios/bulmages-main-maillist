Log:
Modifico la base de datos para que los números de 'pedidos' y
'albaranes' de proveedor tengan un formato alfanumérico. Esto es
importante porque, en muchos casos, los proveedores numeran estos
documentos utilizando un código alfanumérico. En caso de NO especificar
ningún número se marcará con el mismo número que el ID de la entrada en
la base de datos aunque no sea correlativo (en el caso de documentos de
proveedor no es necesario).
