Hola, ya he podido compilar bulmages, y con una kubuntu 8.10.

Los paquetes que he instalado son:


sudo apt-get install build-essential cmake-gui translate-toolkit lokate
sudo apt-get install qt3-assistant qt4-designer qt4-qtconfig qt4-dev-tools
sudo apt-get install postgresql-8.3 postgresql-client libqt4-sql-psql
postgresql-doc-8.3
sudo apt-get install python-minimal python-ooolib python-qt4
python-reportlab python-psycopg2 gettext

Hacer notar que kbabel se llama ahora locate.

...

y el primer bug que encuentro:

en el tray-icon con el boton de la derecha se accede a bulmasetup, pero, no
se inicializa, la razón: en kde 4.2 no existe kdesu, sino que se llama
kdesudo.

Un saludo.
