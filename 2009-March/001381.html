Log:
Creo la clase BusquedaDelegate que es una abstraccion de la clase BusquedaArticuloDelegate pero que a diferencia de esta
esta preparada para funcionar con cualquier tabla y campos que se definan. Hace las busquedas tipo LIKE, muestra el desplegable
y autocompleta la edicion.
Con esta clase se podran eliminar algunos Delegates ya existentes y sera mucho mas sencillo crear delegates para clases nuevas.
De momento solo se esta usando en la clase tutorview.
Las busquedas son tipo LIKE sobre todos los campos configurados y es el primer campo el considerado como valor introducido.

Tambien he hecho que la clase BlSubFormDelegate que unifica las anteriores BCSubFormDelegate y BFSubFormDelegate que ahora derivan de esta y 
que ademas incorpora las llamadas pertinentes a plugins para poder enganchar como delegate elementos de plugin y no solo elementos
declarados en bulmafact, bulmacont o bulmalib. Esto va a permitir quitar algunos archivos mas del core (Aunque por el momento y para
no introducir mas bugs no voy a hacerlo).

Por desgracia Qt considera las clases ItemDelegate de tipo const y no permite alterar ningun valor de la propia clase con lo que
he tenido que hacer un apanyo de variables intermedias. Espero que en proximas versiones de Qt sean algo mas permisivos y se puedan
obviar dichas variables.
