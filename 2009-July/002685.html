Este parche (para trunk) elimina los warnings de compilación del tipo 
siguiente:

/usr/src/packages/BUILD/bulmages/bulmalib/src/blpostgresqlclient.cpp: In 
member function 'int BlPostgreSqlClient::run(QString, int, QString*)':
/usr/src/packages/BUILD/bulmages/bulmalib/src/blpostgresqlclient.cpp:808: 
warning: control reaches end of non-void function
/usr/src/packages/BUILD/bulmages/bulmalib/src/blpostgresqlclient.cpp: In 
member function 'int BlPostgreSqlClient::run(QString, QString, QString, 
QString, QString, QString, QString, QString, QString, QString, QString, 
QString, QString, QString, QString, QString, QString, QString, QString, 
QString, QString)':
/usr/src/packages/BUILD/bulmages/bulmalib/src/blpostgresqlclient.cpp:798: 
warning: control reaches end of non-void function

La importancia de eliminarlos es que este warning no se acepta en el build 
service de openSUSE, e impide la creación del RPM con un mensaje de este 
tipo:

I: Program returns random data in a function
E: bulmages no-return-in-nonvoid-function

Saludos,
Pedro
