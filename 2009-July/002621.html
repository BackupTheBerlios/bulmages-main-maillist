Summary: Elemento predeterminado en campo BlComboBox

Details: Cuando intento hacer que un campo desplegable BlComboBox tenga cierto elemento activo por defecto, al guardar se interpreta como nulo.

El campo está asociado a una consulta (setQuery), a una tabla (setTable) y le digo el campo identificador (setFieldId).

Ejemplo 1:
DESPLEGABLE->setAllowNull( TRUE );
DESPLEGABLE->setId( "1" );

Resultado 1:
Aparece seleccionada inicialmente la línea "--".

Ejemplo 2:
DESPLEGABLE->setAllowNull( FALSE );
DESPLEGABLE->setId( "1" );

Resultado 2:
Aparece seleccionada inicialmente la línea deseada, pero al guardar se interpreta como si estuviera vacío.
