elimino los warnings de compilación del tipo: 
"warning: control reaches end of non-void function"

La importancia de eliminarlos es que este warning no se acepta en el
build service de openSUSE, e impide la creación del RPM con un mensaje
de este tipo:

I: Program returns random data in a function
E: bulmages no-return-in-nonvoid-function
