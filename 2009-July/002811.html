Propongo una solución parecida a como lo hacen los scripts de KDE4. Dos 
partes, en primer lugar un search/replace masivo en todos los CMakeLists.txt, 
cambiando líneas como la siguiente:

	install_targets(/lib bulmalib)
por:
	install(TARGETS bulmalib LIBRARY DESTINATION ${LIB_INSTALL_DIR})

Por otro lado, en el script del directorio raíz, definir una nueva variable 
LIB_SUFFIX que se podrá asignar por los empaquetadores o usuarios para 
ajustar la instalación a las convenciones de cada distro, inicializándola 
con "" en sistemas de 32 bits y con "64" en sistemas de 64 bits. Algo así:

	set(_Init_LIB_SUFFIX "")
	IF(CMAKE_SIZEOF_VOID_P MATCHES "8")
		set(_Init_LIB_SUFFIX "64")
	ENDIF(CMAKE_SIZEOF_VOID_P MATCHES "8")

	set(LIB_SUFFIX "${_Init_LIB_SUFFIX}" CACHE STRING "suffix of lib directory 
name (32/64)")

	set(LIB_INSTALL_DIR "lib${LIB_SUFFIX}")

Es decir, en el momento de ejecutar la configuración, se podría hacer (por 
ejemplo):

	$ cmake .. -DLIB_SUFFIX=32
