> > > Sospecho que bulmasetup v0.11 no se conecta a la base de datos cuando
> > > esta no esta en local y aprovecho para preguntar:
>
> Tienes razon. BulmaSetup solo sirve para configurar bases de datos en
> local. Esto significa que si quieres tener un servidor y algunos clientes y
> quieres configurar con bulmasetup tienes que hacerlo en el servidor y luego
> trasladar los archivos de configuracion a cada uno de los clientes.
>
> Es muy dificil administrar el servidor desde un cliente ya que hay que
> tener privilegios de root para hacerlo de forma senncilla.

Para ejecutar BulmaSetup desde un cliente Linux, yo uso el comando:
ssh -X root en IP bulmsetup

Para conexiones puntuales desde el cliente local al PostgreSQL de otro PC uso:
bulmafact -h IP

(Ver otras opciones ejecutando: bulmafact --help)

>
