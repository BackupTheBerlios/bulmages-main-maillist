Hola,

Aquí os mando una script que genera en cualquier momento
un .orig.tar.gz a partir de un commit actual cualquiera.

Cosas que hace:

-  Genera una versión de paquete que incluye el de CMakeLists.txt +
   timestamp + SHA1 corto de git.

-  Lanza (previa espera a pulsar enter) un tar que recoge todo el
   directorio de trabajo salvo el .git.

Propongo ponerlo en un subdirectorio "tools".  Adjunto además del
fichero un parche de commit de git para ponerlo en la rama master.
