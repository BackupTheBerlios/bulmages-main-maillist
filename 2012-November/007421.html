Algunas empresas -el caso de la nuestra- tienen un programa de puntos establecido, el cual sería intersante usar en BulmaGés, no malpenseis, no digo que os adapteis a X programa de puntos, si no que se realice un plugin que permita ver cuantos puntos tiene cada cliente, cuantos puntos vale cada articulo y demás...

Esto debe figurar al menos en bulmafact y bulmatpv.

Luego el detalle que lo bordaría sería realizar una api en PHP y python mismo, para acceder a ese programa de puntos desde fuera, para integrarlo en tiendas online.
