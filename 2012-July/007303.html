Hola.

Para poner más decimales, sólo tienes que modificar la base de datos y
cambiar el formato de datos a 3 decimales, por ejemplo:


ALTER TABLE lfactura ALTER COLUMN cantlfactura TYPE numeric(12,3);


Con esto podrás escribir 3 decimales en la casilla de cantidad en las
líneas de detalle de las facturas y todo lo demás se configurará para
que salga bien.


Un saludo.
