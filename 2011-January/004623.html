> Hello.
> 
> 
> 1) First you need to download SVN via validated user
> 
> svn checkout \ 
> svn+ssh://developername@svn.berlios.de/svnroot/repos/bulmages/trunk 
> 
> 'developername' is your user name.
> 
> 
> 2) Change file.
> 
> 3) Verify changes with:
> 
> svn status
> 
> - M = modified file.
> - ? = not controled file. If it's new file and you want to add to SVN
> you need to use: 'svn add <file>'
> 
> 4) Commit changes.
> 
> svn commit
> 
> 5) Fill report with vi. 'i' to insert text, then '[esc]:wq' to write
> file and quit.
> 
> (more info: man vi)
> 
> 6) Fill SVN user password. And commit will be send.
>
