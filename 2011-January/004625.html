Well

We suppose that you have the file README.fr in the correct place of your
local downloaded subversion. I assume too that your downloaded subversion is
not the anonymous and is downloaded with your personal account.

If it is a new file you need to add to the repositori, in a shell write

svn add #file#

Note that this only changes your local copy

To see the changes made in your local copy

svn st

And to put changes to all people

svn commit -m "#Write here the comment for the changes made#"


Par example in your case do two commands

svn add README.fr
svn commit -m "I put the README.fr for french translations"
