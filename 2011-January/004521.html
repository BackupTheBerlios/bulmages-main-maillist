I found a way to get languages in ubuntu maverick.
In /usr/share/ there are three "dir" : langpack locale locales
in locale and locales bulmages translation is there.
But in langpack there are mo extension. So I copied bulmages mo
extension there and then the lanzador run french (in translated files of
course !!)

further precisions if needed.
