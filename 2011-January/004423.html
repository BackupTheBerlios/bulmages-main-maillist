Hola Paco

Hace años que no usamos ningún IDE para el proyecto y la verdad es un alivio
no depender del IDE para hacer determinadas cosas.

El proyecto esta montado con cmake y no creo que kdevelop tenga importación
directa de proyectos cmake. Sin embargo desde la consola haces

cd build
cmake ..
sudo make install

Y el solo lo compila todo, monta todo, saca colorines, usa distcc para hacer
compilacion clusterizada, instala. Y funciona mucho mejor que kdevelop,
eclipse o cualquier herramienta "cool" de última generación. La
configuración esta escrita en los archivos CMakeFiles.txt y es fácilmente
entendible y extensible. Para cosas privadas agregas el archivo
CMakeFiles.txt.private y ya queda todo enlazado pero separado y tira millas.

Sobre la edición existen muchos editores, yo últimamente estoy con kate para
edición, rgrep para busquedas y regexxer para sustituciones multiples en
todo el proyecto. Pero es una cuestión personal de gustos (imagino que con
kate te sentirás cómodo ya que tiene mucho en común con kdevelop). Los
puristas aun siguen con el vi vs emacs pero yo no soy tan fino.

Igual parece arcaico pero al final es lo más potente y que mejor funciona ya
que permite una gran flexibilidad. Por ejemplo combinar diferentes fuentes
(subversion, Dropbox, o lo que sea), hacer copy/paste de plugins enteros y
mantiene una visión global de todo el proyecto. Piensa que en un proyecto de
éstas características se combinan repositorios públicos con repositorios
personalizados para cada cliente e incluso algunas carpetas compartidas
entre grupos de desarrolladores sin ser públicas  y se debe mantener una
estructura firme de desarrollo y compilación en lugar de andar duplicando el
proyecto para cada desarrollo o personalización o de lo contrario las tareas
de mantenimiento te comen.

Hay algunas guias en el wiki de iglues (
http://www.iglues.org/wiki_developers/doku.php?id=documentacion:start).
Reconozco que necesita una revisión a fondo pero para empezar puede
aclararte muchas dudas.

Y poca cosa más. Para todo lo que precises pregunta por la lista que te
contestamos encantados.

Salut
