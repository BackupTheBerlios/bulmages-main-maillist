Log:
Agrego al filtro por tipo de columna dos botones: uno para aplicar el filtro y otro para limpiarlo.
El botón de aplicar hace lo mismo que el de actualizar, aunque pone en rojo el filtro en caso de no haber coincidencias.
Para usar este filtro genérico hay que seguir estos pasos:
1) Agregar un QWidget llamado "mui_filtador" y promocionarlo a BlFormListFilter.
2) Agregar en la barra de botones un botón para mostrar y ocultar el widget BlFormListFilter (puede usarse el que hay: "mui_filtrar").
3) Activarle a dicho botón la propiedad checkable y ponerle el icono i_filter.png.
4) Para que el filtro funcione, en el constructor del listado concreto, agregar estas 3 líneas:
 mui_filtrador->setSubFormList ( mui_list );
 connect ( mui_filtrar, SIGNAL ( released ( ) ), mui_filtrador, SLOT ( showHideFilter ( ) ) );
 connect ( mui_filtrador, SIGNAL(aplicar_filtro() ), this, SLOT ( on_mui_actualizar_released() ) );
5) Para que se aplique el filtrado al actualizar el listado concreto, terminar la consulta SQL de presentar() con esto:
 ...WHERE 1 = 1 " + mui_filtrador->generarFiltro() );
