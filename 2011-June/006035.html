Here is a copy from other mail from Daniel.

The first thing to do is update:

1.) git pull


* If we've created a NEW file or directory, they're not in the repository yet. 
We must to add them:

1.) git add <file_or_dir_name>

2.) git commit -a
3.) git push


*. If we want to MOVE or RENAME an existing file or directory we must do:

1.) git mv <old_file_or_dir_name> <new_file_or_dir_name>

2.) git commit -a
3.) git push



*. If we want to DELETE:

1.) git rm <file_or_dir_name>

2.) git commit -a
3.) git push
