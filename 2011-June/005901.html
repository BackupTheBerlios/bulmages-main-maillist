Hola lista.

Veo muchas traducciones en nombres de métodos, y ahora han quedado mensajes de 
depuración que no corresponden al nombre actual.

Por ejemplo:

int BlForm::remove()
{
	blDebug ( "BlForm::borrar", 0 );
...
	blDebug ( "END BlForm::borrar", 0 );
}

¿Y si usamos esta otra forma más genérica que evita tener que estar 
modificando en tantos sitios?

int BlForm::remove()
{
	blDebug ( Q_FUNC_INFO , 0 );
...
	blDebug ( _( "END " ) + Q_FUNC_INFO , 0 );
}

Supongo que no tendría pegas (yo lo uso en mi código).
Un saludo.
