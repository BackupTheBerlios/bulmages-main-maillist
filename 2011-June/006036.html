> 1.) git pull

> * If we've created a NEW file or directory, they're not in the repository

> 
> 1.) git add <file_or_dir_name>

> 3.) git push

> 
> 1.) git mv <old_file_or_dir_name> <new_file_or_dir_name>
> 
> 2.) git commit -a
> 3.) git push
> 
> 
>

> 
> 1.) git rm <file_or_dir_name>
> 
> 2.) git commit -a
> 3.) git push
