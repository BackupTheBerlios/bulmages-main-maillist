Con este "apaño" estás dejando que se conecte cualquiera a la base de datos, 
yo no lo haría.

Prueba con esto antes de abrirte al mundo:
TYPE  DATABASE   		 USER            CIDR-ADDRESS    METHOD
host    all       		  all         		127.0.0.1/32         	 md5

Si tienes que atacar la base de datos desde otra máquina:
host    BD_bulmacont       usuario_bulmacont         0.0.0.0/0             md5

Y en postgresql.conf:

listen_addresses = '*'
