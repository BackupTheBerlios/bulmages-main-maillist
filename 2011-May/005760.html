To allow bulmages to show  up in the systray in unity just enter:

gsettings set com.canonical.Unity.Panel systray-whitelist "['all']"

I on other hand love Unity because I had almost exactly the same setup
with gnome-do and docks in my Gnome classic. 

As for "Ubuntu never coming with Gnome 3" again I think this is a
confusion between Gnome 3 and Gnome Shell, which are not synonyms in
my opinion. Most likely Ubuntu 10.10 will have GTK3 with Unity
recompiled, and with GTK3 it will much easier to have Gnome
Shell. Another thing that may happen is a flavour (as Kubuntu or
Xubuntu) with Gnome shell (I already saw a web page, Gubuntu). 

So never say never! (for those of you that like recursions...)
