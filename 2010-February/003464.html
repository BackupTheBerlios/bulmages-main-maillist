I want to copy 3 values from other table ONLY when id_compo has been changed.

But I always get the error: infinite recursion detected in rules for relation 
"my_table".

It sames like the WHERE condition is never true.

CREATE OR REPLACE RULE actualizar AS
ON UPDATE TO my_table
WHERE new.id_compo IS NOT NULL
AND old.id_compo <> new.id_compo
DO ALSO
	UPDATE my_table SET
		id_compo = new.id_compo,
		coste = t.coste,
		precio_venta = t.precio_venta
	FROM gen.compo AS t
	WHERE new.id_compo = t.id;
