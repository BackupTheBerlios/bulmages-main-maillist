Hola.

Yo suelo usar esto para crear el backup:
sudo pg_dump --format=c --data-only --disable-triggers
--exclude-schema=public --file=<origen>_datos.dump <origen>

Y esto para restaurar sobre una base de datos VACÍA:
sudo pg_restore --format=c --data-only --disable-triggers --verbose
--no-owner --dbname=<destino> <origen>_datos.dump
