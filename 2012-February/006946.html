Veamos, aquí resulta que hay un "pequeño problema".
El sistema de notificaciones que se reciben ahora mismo, no funciona 
como querría.

Se alimenta desde https://gitorious.org/bulmages.atom y eso es algo 
improductivo.

Si examinais el enlace, veréis que solo hay hash, diff, y poca cosa más.

Si vemos el source HTML veremos como hay rutas absolutas y no se 
menciona el servidor, eso causa enlaces muertos en iftt 
(desgraciadamente no solucionables).

Si tuvieramos un servidor (con acceso ssh y cron) podría montar un 
sistema para enviar las notificaciones de manera eficiente, tanto por 
correo como por twitter (que lo intenté, pero me veo con esa traba)

Por otra parte, de mientras intentaré buscar otro sistema y a ver si 
puedo añadir algo más de detalles a los mensajes actuales..
